Build a chatbot with a custom personality — a pirate tutor, a recipe coach, a sarcastic code reviewer. Games are surprisingly vibecode-friendly! Ask Claude to build a canvas-based game — think falling blocks, a platformer, a quiz game, or a card battler. Pure HTML/CSS/JS games need zero server setup. Describe the mechanics you want, playtest in your browser, then ask Claude to tweak physics, add sound effects, or create new levels. It's the fastest creative feedback loop on an iPad.

Dashboards are a great way to make data feel alive. Ask Claude to build a page with Chart.js or D3.js — bar charts, line graphs, pie charts, you name it. Feed it sample data or point it at a public API (weather, stocks, GitHub stats). Add filters, dark mode, and responsive layouts. The whole thing can be a single HTML file deployed to GitHub Pages.

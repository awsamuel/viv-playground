The prompt that started this

I want to test what's possible to build on an iPad by vibecoding from Claude Code. Can you make a colorful interactive website that has buttons for different projects you could consider trying to build from within the iOS Claude app if you have a subscription that includes Claude Code. Offer five big buttons and when you press one, it reveals an animal and then the animal has a little speech bubble that tells you how you might be able to do that kind of project. Each bubble/animal could also suggest a different tool, like Lovable, Replit, Claude or things that I'm not even thinking about. Your goal is to actually create a working prototype and push it to a hosted page on GitHub or some other site where a user would not have to do any set up other than what they do right here in the Claude Code interface.

Inky

Games are surprisingly vibecode-friendly! Ask Claude to build a canvas-based game — think falling blocks, a platformer, a quiz game, or a card battler. Pure HTML/CSS/JS games need zero server setup. Describe the mechanics you want, playtest in your browser, then ask Claude to tweak physics, add sound effects, or create new levels. It's the fastest creative feedback loop on an iPad.
